# Python-3d-video-matplotlib
Create a rotating 3D animated video in Python using Matplotlib. This project generates 3D contour images from mathematical functions and converts them into a GIF video. Includes step-by-step instructions, reusable code, sample setup, and automation to create 180 rotating frames for stunning 3D visual output.
